# 快速入门指南

如果您是第一次使用 VoiSona，以下介绍的是基本的操作方法。

1. [安装 VoiSona](install.md)。
2. 启动 VoiSona。
3. 输入注册时使用的电子邮箱和密码，点击「OK」。
   ![登录画面](images/song_quickstart_Creating_Your_First_Song_1.png)
4. 点击「选择声音」处。  
   声库选择画面将显示出来。
   ![选择声音](images/song_quickstart_Creating_Your_First_Song_2.png)
5. 点击「下载声库」按钮。  
   将开始下载声库。  
   下载完成后，将会显示乐谱编辑画面，并显示所选声库的图片。
   ![下载声库](images/song_quickstart_Creating_Your_First_Song_3.png)
6. 在钢琴卷帘上拖动。  
   将会输入音符。
   ![输入音符](images/song_quickstart_Creating_Your_First_Song_4.png)
7. 双击音符，输入文字后按 <kbd>Enter</kbd> 确认。  
   将会更改歌词。
   ![编辑歌词](images/song_quickstart_Creating_Your_First_Song_5.png)
8. 点击钢琴卷帘上方显示的标尺。  
   将会移动播放位置光标。
   ![点击标尺](images/song_quickstart_Creating_Your_First_Song_6.png)
9. 点击「播放」按钮。  
   会从光标处开始播放乐谱并合成声音。
   ![播放](images/song_quickstart_Creating_Your_First_Song_7.png)
10. 点击「停止」按钮。  
    乐谱会在光标处停止播放。
    ![停止](images/song_quickstart_Creating_Your_First_Song_8.png)
11. 双击钢琴卷帘上方的行，选择要设置的项目。  
    将会设置双击的行所对应的符号。
    ![设置符号](images/song_quickstart_Creating_Your_First_Song_9.png)
12. 点击「参数」按钮。  
    全局参数将显示出来。
13. 拖动「TUNE」「VIA」「VIF」「ALP」「HUS」的滑块进行调整。  
    将会调整影响整首歌曲的参数。
    ![全局参数](images/song_quickstart_Creating_Your_First_Song_10.png)
14. 点击「TMG」标签页。
15. 左右拖动红色线条。  
    将会调整音素的开始和结束时间。
    ![时间调整](images/song_quickstart_Creating_Your_First_Song_11.png)
16. 点击「VOL」「PIT」「ALP」「HUS」中的任一标签页。
17. 自由绘制线条。  
    将会详细调整各个参数。
    ![参数绘制](images/song_quickstart_Creating_Your_First_Song_12.png)
18. 点击「VIB」标签页。
19. 自由绘制线条。  
    将会创建矩形区域。
    ![颤音区域](images/song_quickstart_Creating_Your_First_Song_13.png)
    
    !!! info
        您可以在此区域中调整振幅和周期等。详细操作方法请参阅[调整颤音](adjust-params.md#_11)。

20. 选择菜单栏的「文件」>「导出」>「WAV...」。
    ![导出WAV](images/song_quickstart_Creating_Your_First_Song_14.png)
21. 在设置画面中点击「OK」。  
    WAV 文件将会保存到指定位置。
    ![保存](images/song_quickstart_Creating_Your_First_Song_15.png)
