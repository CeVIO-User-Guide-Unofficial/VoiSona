# 快速入门指南

初次使用 VoiSona 的用户，这里将为您介绍基本的操作方法。

1. 安装 VoiSona。
2. 启动 VoiSona。
3. 输入注册的电子邮箱和密码，点击「OK」。
   ![登录画面](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/299e9bc7-efb1-8035-bc86-f5cfa596f5dd/74970022262300b90deda29220b180d4.png)
4. 点击「选择声音」处。  
   声库选择画面将显示出来。
   ![选择声音](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-8067-b420-e6837b27e0a1/0cd23dbee8515ceece39176193fbb0d5.png)
5. 点击「下载声库」按钮。  
   声库的下载将开始。  
   下载完成后，乐谱编辑画面将显示出来，并显示所选声库的图片。
   ![下载声库](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-8097-b972-f379df10b51b/de740f633734d8099f4ce89217b32055.png)
6. 在钢琴卷帘上拖动。  
   音符将被输入。
   ![输入音符](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-80a4-8260-fb712a6f745b/156069d2d68a55661159513f08f1ab5e.png)
7. 双击音符，输入文字后按 Enter 确认。  
   歌词将被更改。
   ![编辑歌词](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-8062-b135-f02fa7a38f08/6e063805d5f06f2a88c917f6c10ab2ec.png)
8. 点击钢琴卷帘上方显示的标尺。  
   播放位置光标将移动。
   ![点击标尺](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-80b9-be4b-d066bd1e63fd/1dbed5fdb3067bb5f996e435baed2832.png)
9. 点击「播放」按钮。  
   从播放位置光标处开始播放乐谱并合成声音。
   ![播放](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-8070-81d9-e0a90c40619e/b052aeb4bf4eccefd8361e5749d1ff42.png)
10. 点击「停止」按钮。  
    乐谱在播放位置光标处停止播放。
    ![停止](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-8081-a825-fd281f333b6f/0ee88e4be1253591941eebbbc31285a2.png)
11. 双击钢琴卷帘上方的行，选择要设置的项目。  
    双击的行所对应的符号将被设置。
    ![设置符号](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-802a-8579-c8a1ad5dbbba/3f727ae456bf8455ac13205f77e5a0bb.png)
12. 点击「参数」按钮。  
    全局参数将显示出来。
13. 拖动「TUNE」「VIA」「VIF」「ALP」「HUS」的滑块进行调整。  
    影响整首歌曲的参数将被调整。
    ![全局参数](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-807f-afa2-f3aae28c7195/40f29d947a2dbc1776628f9627f392d1.png)
14. 点击「TMG」标签页。
15. 左右拖动红色线条。  
    音素的开始和结束时机将被调整。
    ![时机调整](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-8027-99fc-f7b1ba33049d/a971dee0bc79ef304457eaa8bae364be.png)
16. 点击「VOL」「PIT」「ALP」「HUS」中的任一标签页。
17. 自由绘制线条。  
    各个参数将被详细调整。
    ![参数绘制](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-800b-963a-f4c4570f5396/1990840f50860999913aebb485a55030.png)
18. 点击「VIB」标签页。
19. 自由绘制线条。  
    矩形区域将被创建。
    ![颤音区域](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-808f-bd1e-e4941a09bd20/96328c01e20c7b2409fb0ac498688a1a.png)

    > 您可以在此区域中调整振幅和周期等。详细操作方法请参阅[调整颤音](adjust-params.md#_11)。

20. 选择菜单栏的「文件」>「导出」>「WAV...」。
    ![导出WAV](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-803b-bfc9-d1abb277481d/38a10d67e1171e775275c5fc4366c142.png)
21. 在设置画面中点击「OK」。  
    WAV 文件将保存到指定位置。
    ![保存](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-80a8-a6be-ef62546bd3a5/31df09145e87635796727d87a923acd3.png)
