# 更改环境设置

可以更改语言设置、编辑器整体行为以及快捷键分配等。

1. 选择以下操作之一：
    - **独立应用**：菜单栏的「编辑」>「环境设置」
    - **乐器插件、ARA 插件**：点击「菜单」按钮后选择「环境设置」
      ![插件设置1](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bee9bc7-efb1-8045-b58a-cff24a787c5e/63463f9c6865c84260a804aaa1916e02.png)
      ![插件设置2](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-80a3-83d1-cdd5c1e67d0c/1e1c522da174858750b32bb2b35adeda.png)
2. 切换左侧标签页，进行各种设置。
   ![设置画面](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bee9bc7-efb1-801b-98f6-f6a6bb6867ad/1d96336890913fe0c6112f920cd7f87c.png)

> 在「快捷键」设置画面中，可以自由更改各操作分配的按键。善用快捷键可以大幅提高工作效率，请务必利用。
> ![快捷键设置](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bee9bc7-efb1-8092-bc9f-fc78b35cf93d/7a55642de6ea30ad0c5a7edd046ae29f.png)
