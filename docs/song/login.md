# 登录

您需要先登录才能使用 VoiSona（PC版）。登录后即可使用您已购买的声库。

!!! info
      首次启动应用时，将自动显示登录画面。

1. 选择以下操作之一：
    - **独立运行**：菜单栏的「帮助」>「登录」
    - **乐器插件、ARA 插件**：点击「菜单」按钮后选择「登录」
      ![插件菜单登录](images/song_login_Signing_In_1.png)
      ![插件菜单2](images/song_login_Signing_In_2.png)
2. 输入注册时使用的电子邮箱和密码，然后点击「OK」。  
   登录成功后，登录画面将关闭。
   ![登录画面](images/song_login_Signing_In_3.png)
