# 登录

您需要先登录才能使用 VoiSona（PC版）。登录后即可使用您已购买的声库。

!!! info
      首次启动应用时，将自动显示登录画面。

1. 选择以下操作之一：
    - **独立运行**：菜单栏的「帮助」>「登录」
    - **乐器插件、ARA 插件**：点击「菜单」按钮后选择「登录」
      ![插件菜单登录](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-8066-ba53-f44cd325a64b/7cd494dad799f6f2d0bd32df69870dce.png)
      ![插件菜单2](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-80d2-9e42-de5c12d1c318/0255d41c8b9e381f672af2b822622f59.png)
2. 输入注册时使用的电子邮箱和密码，然后点击「OK」。  
   登录成功后，登录画面将关闭。
   ![登录画面](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-8090-9dbe-fdd301314f5d/d22c44f7315c0e19ec02fda523cfb6bd.png)
