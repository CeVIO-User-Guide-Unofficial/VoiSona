# 安装 VoiSona

要使用 VoiSona，请先安装应用程序。更新时也可按照相同的步骤进行。

!!! warning
      请在电脑连接互联网的状态下进行操作。

1. 访问 DOWNLOAD 页面并登录。
2. 点击「Windows版」或「macOS版」按钮。
   ![下载按钮](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-8063-8e6a-e67e2505e905/3920c66e683e1d88f6153a4b67cdf633.png)
3. 最新版本的安装程序将被下载。
4. 打开下载的安装程序。
5. 选择安装过程中使用的语言，点击「OK」。
   ![选择语言](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-80b5-ac2a-c304e0227e93/bd2298948dbc07cdb4c9656f8e925aaa.png)
6. 确认使用许可协议后，如同意则勾选「同意」并点击「下一步」。
   ![许可协议](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-80dd-90ca-dbc6d86fb8df/8d3c0a216b04fc1dfd6642dfbfbabf22.png)
7. 如需要在桌面创建图标请勾选，然后点击「下一步」。
   ![创建图标](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-80dd-b072-e069e4cb6c16/0fb51359d60ff533cb800a4649f44e1d.png)
8. 点击「安装」。安装将开始。
   ![安装](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-8062-9c8e-f1172855ad4d/2c8db1aa71e5a1e17fe6a945e90c06c0.png)
9. 安装完成后，点击「完成」。
   ![完成](https://s3.ap-northeast-1.amazonaws.com/wraptas-prod/voisona-manual/2bde9bc7-efb1-80bd-ae10-f32f6dcabdff/4fd93d32efd1b18946d9f79f4b813732.png)

## 安装目标文件夹

| 类型 | Windows | macOS |
|------|---------|-------|
| 安装目录 | `C:\Program Files\Techno-Speech` | `/Users/<用户名>/Library/Techno-Speech` |
| 插件目录 (VST3) | `C:\Program Files\Common Files\vst3\Techno-Speech` | `/Library/Audio/Plug-Ins/VST3/Techno-Speech` |
| 插件目录 (AU) | — | `/Library/Audio/Plug-Ins/Components` |
| ARA 插件目录 | `C:\Program Files\Common Files\ARA` | `/Library/Audio/Plug-Ins/ARA` |
