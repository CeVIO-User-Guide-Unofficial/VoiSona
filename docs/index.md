# VoiSona 用户指南

![VoiSona](image.png)

**来吧，将想象化为歌声，以双手赋予文字生命。**

VoiSona 是基于最新的 AI 技术，逼真地再现人声的声音合成软件系列。

- VoiSona Song 能合成高质量的歌声。支持 Windows、macOS、iOS 和 iPadOS 平台，可作为 VSTi/Audio Units (AU) 插件在 DAW 中使用，也可独立运行。
- VoiSona Talk 能合成高质量的朗读语音。支持 Windows 和 macOS 系统。

本网站为 VoiSona 用户指南的非官方译文，因为[官方手册](https://manual.voisona.com/)虽然详尽，但仅提供了日语和英语版本。

本网站不隶属于 Techno-Speech。

- [Talk 用户指南](talk/manual.md) — 朗读编辑器的完整使用说明
