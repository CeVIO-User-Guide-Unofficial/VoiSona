# 登录

您需要先登录才能使用 VoiSona Talk。登录后即可使用您已订阅的声库。

!!! info
      首次启动应用时，将自动显示登录画面。

1. 点击「菜单」按钮，选择「帮助」>「登录」。
   ![登录菜单](images/talk_login_c6b0f588.png)
2. 输入注册时使用的电子邮箱和密码，然后点击「OK」。  
   登录成功后，登录画面将关闭。
   ![登录画面](images/talk_login_8ae2a398.png)
