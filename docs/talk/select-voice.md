# 选择声库

可以从声库中选择一个来朗读台词。

## 下载并选择声库

可以从已订阅的声库中选择您喜欢的版本使用。

1. 点击显示「选择声库」的区域。
   ![选择声库](images/talk_select-voice_e1c305c8.png)
2. 将光标对准想要使用的声库。
3. 点击想要使用的版本对应的「下载声库」按钮。  
   声库下载将开始。  
   下载完成后，声库选择画面将关闭。
   ![下载声库](images/talk_select-voice_d8e05a0b.png)

!!! info
      首次下载且未拥有声库时，下载后会自动被选中。

4. 再次打开声库选择画面，点击版本号。  
   乐谱编辑画面将显示出来，并显示所选声库的图像。
   ![选择版本](images/talk_select-voice_f6e1546c.png)
   ![声库已选择](images/talk_select-voice_ae73ae6d.png)

!!! info
      如果「下载声库」按钮未显示  
      点击显示的按钮前往 [DOWNLOAD](https://voisona.com/talk/download/) 页面，将 VoiSona Talk 更新到最新版本。  
      更新后重启 VoiSona Talk，「下载声库」按钮将会出现。  
      请注意，如果您运行的 VoiSona Talk 版本过旧且不支持相关声库，「下载声库」按钮将不会显示。
      ![下载按钮未显示](images/talk_select-voice_7db8593d.png)

## 添加声库

可以按照以下步骤添加声库。

1. 声库可从官方网站的 [ARTIST](https://voisona.com/talk/artist/) 页面购买。
2. 重启 VoiSona Talk 应用。

!!! info
      当您[登录](login.md)拥有声库授权的账户后，声库将自动添加到可用声库列表中。

3. [下载并选择声库](#_1)。
