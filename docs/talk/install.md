# 安装 VoiSona Talk

要使用 VoiSona Talk，请先安装应用程序。更新时也可按照相同的步骤进行。

!!! info
      请在电脑连接互联网的状态下进行操作。

1. 访问 [DOWNLOAD](https://voisona.com/talk/download/) 页面并登录。
2. 点击「Windows版」或「macOS版」按钮。  
   最新版本的安装程序将被下载。
   ![下载按钮](images/talk_install_b25e2f82.png)
3. 打开下载的安装程序。
4. 选择安装过程中使用的语言，点击「OK」。
   ![选择语言](images/talk_install_8298c2c4.png)
5. 确认使用许可协议后，如同意则勾选「同意」并点击「下一步」。
   ![许可协议](images/talk_install_a6a48865.png)
6. 如需要在桌面创建图标请勾选，然后点击「下一步」。
   ![创建图标](images/talk_install_ec62ed1e.png)
7. 点击「安装」。  
   安装将开始。
   ![安装](images/talk_install_ffa64fc1.png)
8. 安装完成后，点击「完成」。
   ![完成](images/talk_install_6e18b99b.png)

!!! info
      安装目标文件夹
      - **Windows**：`C:\Program Files\Techno-Speech`
      - **macOS**：`/Users/<用户名>/Library/Techno-Speech`
