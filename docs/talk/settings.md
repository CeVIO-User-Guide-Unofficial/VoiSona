# 更改环境设置

可以更改语言设置、编辑器整体行为以及快捷键分配等。

1. 点击「菜单」按钮，选择「编辑」>「环境设置」。
   ![环境设置菜单](images/talk_settings_b125e7d3.png)
2. 切换标签页，进行各种设置。

!!! info
      在「默认值」中可以更改默认导出名称。  
      另请参阅[替换导出名称中的字符串](./import-export.md#_8)。
      ![默认导出名称](images/talk_settings_bd89c620.png)

!!! info
      在「快捷键」设置画面中，可以自由更改各操作分配的按键。

      善用快捷键可以大幅提高工作效率，请务必利用。
      ![快捷键设置](images/talk_settings_cb171a3f.png)
