# 快速入门指南

本节介绍 VoiSona Talk 初次使用时的基本操作流程。

1. [安装 VoiSona Talk](install.md)。
2. 启动 VoiSona Talk。
3. 输入注册时使用的电子邮箱和密码，然后点击「OK」。
   ![登录画面](images/talk_quickstart_5350c6d9.png)
4. 点击显示「选择声库」的区域。  
   声库选择画面将显示出来。
   ![选择声库](images/talk_quickstart_2eab3dd0.png)
5. 点击想要使用的声库对应的「下载声库」按钮。  
   下载完成后，声库图像将显示出来。
   ![下载声库](images/talk_quickstart_7d33248f.png)
6. 在「台词」列的单元格中点击，输入台词后按 Enter 键。  
   台词将被添加。
   ![输入台词](images/talk_quickstart_7f483203.png)
7. 点击轨道时间轴的上部。  
   播放位置光标将移动。
   ![移动光标](images/talk_quickstart_d8684dc0.png)
8. 点击「播放」按钮。  
   从播放位置光标处开始播放音频。
   ![播放](images/talk_quickstart_abedefbd.png)
9.  点击「暂停」按钮。  
   音频将在播放位置光标处暂停。
   ![暂停](images/talk_quickstart_bc21b8c4.png)
10. 点击「参数」按钮。  
   全局参数面板将显示出来。
11. 拖动「SPD」「VOL」「PIT」「INTO」「ALP」「HUS」的滑块。  
   这些参数将应用于整个台词。
   ![全局参数](images/talk_quickstart_dda69d1c.png)
12. 在「ACC」调整画面中点击莫拉。  
   重音位置将被更改。
   ![重音调整](images/talk_quickstart_026eb256.png)
13. 在发音编辑区域编辑假名。  
   发音（读法）将被更改。
   ![发音编辑](images/talk_quickstart_c3e2abd9.png)
14. 点击「STY」标签页。
15. 在调整画面中点击指定位置，然后拖动风格图表进行移动。  
   台词的风格将被精细调整。
   ![风格调整](images/talk_quickstart_cdd3593e.png)
16. 点击「VOL」「PIT」「ALP」「HUS」之一的标签页。
17. 选择画笔工具，自由绘制线条。  
   各参数将被精细调整。
   ![参数绘制](images/talk_quickstart_4bbb3545.png)
18. 点击「菜单」按钮，选择「文件」>「导出」>「导出混音 WAV 文件...」。
   ![导出菜单](images/talk_quickstart_4b0e425e.png)
19. 选择保存位置，输入文件名，点击「保存」。  
   WAV 文件将保存到指定位置。
